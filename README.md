# HackPad-MicroMacro
Blueprint project of a hackpad to be used with my computer for gaming and audio work.

BOM:
- 4x Cherry MX Switches
- 12x SK6812 MINI LEDs
- 1x XIAO RP2040
- 4x Blank DSA Keycaps (White)
- 4x M3x16 Bolts
- 4x M3 Heatset Inserts
- 1x Printed Base
- 1x Printed Lid
